# SafeStep 실행 순서 (팀원용)

> 서버는 준범이 켜둡니다. 팀원은 **앱 세팅만** 하면 됩니다.

---

## 팀원 — 처음 세팅하는 경우

### 1단계 — 레포 클론

```powershell
git clone https://github.com/milk152milk/navigation.git
```

> 클론하면 모델 파일(bbox.pt, surface.pt)을 포함한 모든 파일이 자동으로 다운로드됩니다.

---

### 2단계 — SERVER_URL 입력

준범에게 현재 서버 URL을 받아서 아래 파일을 수정합니다:

```
navigation/준범/SafeStep/app/src/main/java/com/safestep/app/detect/RemoteDetector.kt
```

27번째 줄 수정:

```kotlin
const val SERVER_URL = "https://준범이_알려준_URL/detect"
```

---

### 3단계 — 앱 빌드 및 실행

Android Studio에서 아래 프로젝트를 엽니다:

```
navigation/준범/SafeStep
```

폰을 USB로 연결한 뒤 상단 **▶ Run** 버튼 클릭 (또는 `Shift+F10`)

앱 최초 실행 시 아래 권한을 모두 **허용** 해주세요:
- 카메라
- 마이크
- 위치

---

### 4단계 — 연결 확인

폰 브라우저에서 아래 주소 접속:

```
https://준범이_알려준_URL/health
```

아래처럼 뜨면 정상입니다:

```json
{"status": "ok", "yolo": "bbox.pt", "surface": "surface.pt"}
```

---

## 팀원 — 매번 실행할 때

```
① 준범에게 현재 서버 URL 확인
② URL 바뀌었으면 RemoteDetector.kt 수정 후 앱 재빌드
③ 앱 실행
```

> 준범이 ngrok 고정 도메인을 설정해두면 URL이 항상 동일해서 재빌드 불필요합니다.

## 준범 — 서버 실행 (서버 담당)

### 처음 세팅 (최초 1회)

Python 3.9 이상 필요.

```powershell
pip install fastapi uvicorn ultralytics opencv-python pillow numpy transformers torch pyngrok python-dotenv
```

---

### 서버 시작

```powershell
cd navigation/준범/SafeStep/server
python server.py
```

정상 시작 시 아래와 같이 출력됩니다:

```
[SafeStep] YOLO 탐지 로드 완료 ✅
[SafeStep] 노면 세그멘테이션 로드 완료 ✅
[SafeStep] 거리 추정 모델 로드 완료 ✅
============================================================
[SafeStep] 📱 공개 URL (ngrok): https://xxxx-xxxx.ngrok-free.app
[SafeStep] RemoteDetector.kt 에 아래 URL 입력:
  SERVER_URL = "https://xxxx-xxxx.ngrok-free.app/detect"
============================================================
INFO: Uvicorn running on http://0.0.0.0:8000
```

이 URL을 팀원들에게 공유하세요.

---

### 서버 종료

터미널에서 `Ctrl + C`

강제 종료 (터미널이 닫혔거나 응답 없을 때):

```powershell
Get-Process python | Stop-Process -Force
```

---

### ngrok 고정 도메인 설정 (선택, URL 고정)

설정 시 서버를 재시작해도 URL이 바뀌지 않아 팀원들이 매번 재빌드할 필요가 없습니다.

1. [dashboard.ngrok.com](https://dashboard.ngrok.com) → **Domains** → **New Domain** → 무료 도메인 생성
2. `server/.env` 수정:
   ```
   NGROK_AUTH_TOKEN=토큰값
   NGROK_DOMAIN=발급받은도메인.ngrok-free.app
   ```

---

### 방화벽 설정 (연결 안 될 때 1회, 관리자 PowerShell)

```powershell
New-NetFirewallRule -DisplayName "SafeStep Server" -Direction Inbound -Protocol TCP -LocalPort 8000 -Action Allow
```

## 깃 업데이트 방법 (준범)

코드 변경 후 아래 순서로 커밋·푸시합니다:

```powershell
cd navigation/준범/SafeStep

# 변경 내용 확인
git status

# 전체 추가
git add .

# 커밋 (메시지는 자유롭게)
git commit -m "기능 업데이트: 계단 감지, 보행 모드, 배터리 절약 등"

# 푸시
git push origin main
```

팀원은 최신 코드를 받으려면:

```powershell
cd navigation
git pull origin main
```
